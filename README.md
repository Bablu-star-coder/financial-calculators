# 📈 Glassmorphism Financial Calculators

A beautiful, premium, full-stack financial calculator suite built to mirror industry-grade analytics dashboards.

## ✨ Features
* 📊 **SIP Calculator:** Track monthly systematic investment plans matching Groww calculations down to the single rupee.
* 💸 **SWP Engine:** Evaluate customized asset withdrawal pipelines cleanly.
* 🎈 **Inflation SIP Model:** Adjust prospective earnings parameters cleanly based on real economic inflation rates.
* 🎨 **Premium UI:** Glassmorphism dashboard frame styled completely with custom range slider mechanics.

## 🛠️ Stack Architecture
* **Frontend UI:** HTML5, CSS3 Grid layouts, Vanilla JavaScript Engine
* **Visual Graphics Engine:** Chart.js (Interactive Doughnut Layers)
* **Backend Framework:** Node.js, Express Application Gateway
* **View Engine Compilers:** EJS (Embedded JavaScript Templates)
